# Dotfiles

Hyprland rice — Catppuccin Mocha (Green accent)

## İçerikler

| Klasör | Açıklama |
|--------|----------|
| hypr | Hyprland, hyprlock, hypridle, hyprswitch |
| waybar | Üst panel |
| rofi | Uygulama başlatıcı & güç menüsü |
| kitty | Terminal |
| swaync | Bildirim servisi |
| fastfetch | Sistem bilgisi |
| kvantum | Qt tema ayarı |
| networkmanager-dmenu | WiFi yönetimi |

## Kurulum

```bash
git clone https://github.com/nazms23/dotfiles.git ~/dotfiles
cd ~/dotfiles
# Her klasör için symlink oluştur:
ln -sf ~/dotfiles/hypr/* ~/.config/hypr/
ln -sf ~/dotfiles/waybar/* ~/.config/waybar/
ln -sf ~/dotfiles/rofi/* ~/.config/rofi/
ln -sf ~/dotfiles/kitty/* ~/.config/kitty/
ln -sf ~/dotfiles/swaync/* ~/.config/swaync/
ln -sf ~/dotfiles/fastfetch/* ~/.config/fastfetch/
ln -sf ~/dotfiles/kvantum/* ~/.config/Kvantum/
ln -sf ~/dotfiles/networkmanager-dmenu/* ~/.config/networkmanager-dmenu/
```
